## 📋 Sobre o Projeto

**Sistema de Eleição** é uma plataforma completa para condução de eleições com sincronização em tempo real entre múltiplas telas, garantindo transparência e agilidade no processo eleitoral.

> ⚠️ Este repositório é uma vitrine do projeto. O código-fonte é mantido em repositório privado.

---

## 🚀 Funcionalidades

### 🖥️ Tela 1 — Organizador (Admin)
- Painel de controle completo da eleição
- Gerenciamento de candidatos e votantes
- Chamada nominal dos votantes
- Controle de abertura e encerramento de votação
- Acompanhamento em tempo real dos votos

### 📽️ Tela 2 — Projetor (Público)
- Exibição em telão para acompanhamento do público
- Mostra as etapas da eleição em tempo real
- Exibe o nome do votante sendo chamado
- Resultados parciais e finais com visual atrativo

### ✅ Tela 3 — Votação
- Interface limpa e intuitiva para o votante
- Voto individual e seguro
- Confirmação de voto
- Sincronização instantânea com as demais telas

---

## ⚡ Destaque Técnico

- **Real-time** — Todas as 3 telas sincronizadas via WebSocket
- **Multi-tela** — Cada tela tem função e interface independente
- **Transparência** — Público acompanha todo o processo ao vivo
